# Homemade bags Backend

This is the Backend for Homemade Bags Project

### Features

- Add new Product
- See all your Products
- Update a Specific Product
- Delete a Specific Product

### Technologies Used

- NodeJs
- ExpressJs
- Mongodb

## Installation Steps

- To Install dependencies run below commands

```Bash
    npm install
```

- To Start Backend

```Bash
    node Server.js
```
